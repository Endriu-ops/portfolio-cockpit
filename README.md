# Portfolio Cockpit Pro

App pronta per Streamlit Cloud.

## Deploy

1. Crea un repository GitHub.
2. Carica tutti i file dello ZIP.
3. Vai su https://share.streamlit.io
4. Scegli il repository.
5. Main file: `app.py`
6. In Settings > Secrets inserisci le chiavi da `.streamlit/secrets.toml.example`.

## Avvio locale

```bash
pip install -r requirements.txt
streamlit run app.py
```

Nota: strumento educativo, non consulenza finanziaria.
